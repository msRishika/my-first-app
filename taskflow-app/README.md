# TaskFlow - Day 2 Core App

## Features

### Authentication
- Login page with email and password
- User stored in localStorage
- Redirect to Dashboard after login
- Logout functionality

### CRUD Operations
- Create Task
- Read Tasks
- Update Tasks
- Delete Tasks

### Routes
- / → Login Page
- /dashboard → Dashboard

### Tech Stack
- React
- Vite
- React Router DOM

## Run Locally

npm install

npm run dev
